# Tumble game-play demo
![tumble demo](tumble.jpg)
HyperGodot Demo of a Third-Person Controller.  

# Controls
* wsad for movement 
* left mouse click for grappling hook 
* "x" for light toggle
* space bar for jump
* "r" to respawn
